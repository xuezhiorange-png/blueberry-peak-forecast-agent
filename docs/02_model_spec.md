# 预测模型技术规范

## 1. 目标变量

每日有效商品果到厂量：

- 时间范围：1—4月；
- 默认剔除普鲜、普青、普冻、废果；
- 默认剔除巴松加工厂；
- 重量单位统一为吨。

持续峰值：

\[
P_{stable}=\max_t median(Y_{t-1},Y_t,Y_{t+1})
\]

单日峰值：

\[
P_{raw}=\max_t Y_t
\]

## 2. 四层模型

### 2.1 规模层

对分场 s、品种 v：

\[
V_{s,v}=A_{s,v}\times q_{s,v}\times r_{s,v}
\]

- A：农场/分场种植面积；
- q：预计亩产；
- r：有效商品果率；
- V：预计进入指定加工厂的有效商品果总量。

加工厂建筑面积不进入模型。

### 2.2 自然成熟层

\[
M_t=\sum_{s,v}V_{s,v}\times G_{r,v}(\tau_t-\delta_{s,v})
\]

- G：区域—品种共享成熟曲线；
- τ：由积温、辐射、降雨、低温等形成的物候时间；
- δ：海拔、棚型、树龄、修剪、花期造成的偏移。

### 2.3 采摘与积压层

\[
Y_t=\min(H_t,M_t+B_{t-1}\lambda_t)
\]

\[
B_t=\max(0,B_{t-1}+M_t-Y_t)
\]

\[
H_t=N_t\times productivity_t\times efficiency_t
\]

春节主要作用于可用人数 N，不直接改变自然成熟量 M。

### 2.4 残差修正层

使用梯度提升模型修正结构模型残差，特征包括：

- 最近3/7/14日实际入库；
- 计划完成率和剩余量；
- 7/14/21日积温及变化；
- 降雨和连续阴雨；
- 采摘人数和人均效率；
- 春节距离、节前/节中/节后标签；
- 品种HHI、分场HHI、有效分场数；
- 多品种成熟重叠指数；
- 积压库存估计。

## 4. 不确定性

- P50：中心预测；
- P80/P90：通过残差分位数或 conformal prediction 生成；
- 同时分解总量不确定性、日期偏移不确定性和采摘能力不确定性。

## 5. 防止数据泄漏

回测某日预测时，仅可使用当日之前已存在的：

- 产量计划版本；
- 实际入库；
- 已观测天气和当时可获得的天气预报；
- 人员计划；
- 物候记录。

不得使用最终产季总量、最终品种构成或预测日之后的实际数据。
