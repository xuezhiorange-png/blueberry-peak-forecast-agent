## 1. 报告首页结论

- 农场位置与识别坐标；
- 品种定植面积汇总；
- 预计有效商品果总量 P50/P80/P90；
- 持续峰值 P50/P80/P90及日期；
- 单日操作峰值及日期；
- 高峰持续期；
- 可信度等级及历史回测误差。

## 4. 计算说明模板

### 输入

列出用户输入的地点和各品种面积。

### 自动补充参数

逐项列出系统推断值、P80区间、数据来源和样本量。

### 计算过程

按“总量估计→自然成熟曲线→气候移动/压缩→采摘实现→多品种叠加→峰值提取”说明。

### 准确性

列出模型在同区域、同品种和全体样本上的回测结果，说明当前预测属于规划模式还是动态模式。

### 风险与建议

按高峰日期、春节、人员、预冷、班次、分流和品种错峰输出可执行建议。
